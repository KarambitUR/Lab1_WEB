Розв'язання прямокутного трикутника за допомогою JavaScript

Відкрийте консоль браузера (F12 → Console) та викличте функцію triangle(value1, type1, value2, type2) для розв'язання прямокутного трикутника.

Допустимі типи: leg, hypotenuse, adjacent angle, opposite angle, angle. Кути задаються у градусах.

Приклади викликів:

triangle(3, "leg", 4, "leg")
triangle(4, "leg", 8, "hypotenuse")
triangle(5, "hypotenuse", 45, "angle")
triangle(30, "adjacent angle", 5, "leg")
triangle(60, "opposite angle", 7, "leg")
